# 🛡️ CyberAI Defense
## Generative AI as a Cyber Crime Psychologist

Educational research prototype for predictive cyber defense.

### Overview

CyberAI Defense models attacker behavior as a sequence of actions across the MITRE ATT&CK kill chain. Instead of detecting attacks after they occur, the system predicts the next attacker move and enables proactive defense.

### Features

- Attacker Behavior Modeling
- Threat Prediction
- Threat Scoring
- MITRE ATT&CK Mapping
- Explainable AI (XAI)
- Threat Intelligence Enrichment
- Streamlit Dashboard
- FastAPI REST API
- Attack Simulation Engine

### Technology Stack

- Python
- Streamlit
- FastAPI
- NumPy
- Scikit-Learn
- Plotly
