## Purpose
Load public cybersecurity datasets, standardise their schema,
and map raw attack labels to MITRE ATT&CK stages.

## Supported Datasets
| Dataset    | Source                                             | Labels |
|------------|----------------------------------------------------|--------|
| CICIDS2017 | https://www.unb.ca/cic/datasets/ids-2017.html      | 15     |
| UNSW-NB15  | https://research.unsw.edu.au/projects/unsw-nb15-dataset | 9  |
| Synthetic  | Generated by `load_synthetic()` — no download      | 9      |

## Folder Layout
```
data/
├── raw/           ← Place downloaded CSVs here
├── processed/     ← Standardised outputs (auto-generated)
├── sequences/     ← Attack sequences (created by Module 3)
└── dataset_loader.py   ← This module
```

## Quick Start
```python
from data.dataset_loader import DatasetLoader

loader = DatasetLoader()

# Option A — synthetic demo (no downloads needed)
df = loader.load_synthetic(n_samples=5000)

# Option B — real CICIDS2017
# df = loader.load_cicids("Friday-WorkingHours-Afternoon-DDos.pcap_ISCX.csv")

df    = loader.standardise(df)
stats = loader.explore(df)
loader.save(df, "processed_data.csv")
```
